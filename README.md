# little-starwars-project

## Tech Stack

- React
- Typescript
- styled-components
- react-router
- Material-UI

## Available Scripts

| Script    | Description                                          |
| --------- | ---------------------------------------------------- |
| `install` | Installs packages that project relay on.             |
| `start`   | Runs the app in the development mode.                |
| `test`    | Launches the test runner.                            |
| `build`   | Builds the app for production to the `build` folder. |
